# ReLeet

Chrome extension for saving and revising failed LeetCode problems.

## Features

- Save failed LeetCode problems
- Smart revision workflow
- Future spaced repetition support
- Chrome extension built with React + Vite

## Tech Stack

- React
- TypeScript
- Vite
- CRXJS
- Chrome Extension Manifest V3

## Current Status

🚧 In Development

Currently working on:
- Detecting failed submissions on LeetCode
- Injecting revision UI dynamically

## Setup

```bash
npm install
npm run dev
```
